# Campus Bus Tracking System

This project is a web-based application for tracking campus buses in real-time.

## Features
- Real-time bus tracking
- ETA calculation
- Crowd monitoring (Low/Medium/High)
- Interactive map using Leaflet.js

## Technologies Used
- HTML, CSS, JavaScript
- Leaflet.js
- Node.js, Express
- MongoDB

## How to Run
1. Start MongoDB
2. Run backend:
   cd Server
   node app.js
3. Open frontend:
   Client/index.html
